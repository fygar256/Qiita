linux用バイナリファイルエディタの設定ファイル

バイナリファイルをいじるとき、bviを持ち出すのだが、デフォルトの設定ではファイルに書き込み禁止になっていて、書き込みスィッチをよく忘れてweb検索してしてしまうので、覚書＆設定ファイルに設定。

~/.bvircというファイルに、書いておけばいい。
```.bvirc
set memmove # 書き込み可にする
```
